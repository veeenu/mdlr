# mdlr

A barebones markdown live-reload viewer.

## Install

```
cargo install --git https://github.com/veeenu/mdlr
```

## Run

```
$ ls
README.md*  test.md*
$ mdlr
```

Now navigate to http://localhost:8000/README.md.
That's it!
